## 常用的位操作

1. 利用或操作 `|` 和空格将英文字符转换为小写

   > ```
   > ('a' | ' ') = 'a'
   > ('A' | ' ') = 'a'
   > ```

2. 利用与操作 `&` 和下划线将英文字符转换为大写

   > ```
   > ('b' & '_') = 'B'
   > ('B' & '_') = 'B'
   > ```

3. 利用异或操作 `^` 和空格进行英文字符大小写互换

   > ```
   > ('d' ^ ' ') = 'D'
   > ('D' ^ ' ') = 'd'
   > ```

4. 判断两个数是否异号

   > ```
   > int x = -1, y = 2;
   > bool f = ((x ^ y) < 0); // true
   > 
   > int x = 3, y = 2;
   > bool f = ((x ^ y) < 0); // false
   > ```

5. n&(n-1)  作用是消除数字 n 的二进制表示中的最后一个 1。
